This folder contains a 1-dimensional synthetic data for gaussian process regression. In the train.txt file provided, each row is a separate training example; the first column is a feature, while the second column has the real-valued labels.
In the test.txt file provided, each row is a test instance.

Files:
train.txt - Training instances + labels (50 x 2)
test.txt - Test instances (1000 x 1)
