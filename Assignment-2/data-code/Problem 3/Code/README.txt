In this directory, the following files are provided.

 1. LogisticRegression.m  -  (Template) This function implements (unregularized) logistic regression
 2. LogisticRegressionL2.m  -  (Template) This function implements L2-regularized logistic regression
 4. classification_error.m  -  This functions computes the classification error given the predicted and true labels.
 
 Use fminunc MATLAB function for solving the logistic regression optimization problem.
