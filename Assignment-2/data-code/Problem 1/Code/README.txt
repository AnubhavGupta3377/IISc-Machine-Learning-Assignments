In this directory, the following files are provided.

 1. LSR.m  -  (Template) This function implements (linear) least squares regression
 2. LSRL2.m  -  (Template) This function implements L2-regularized (linear) least squares regression
 3. plot_script.m - This function plots the ridge regression curve for Problem 1a
 4. mean_squared_error.m  -  This functions computes the mean squared error given the predicted and true labels.
