This folder contains a 250-dimensional synthetic data for regression. In the data files provided, each row is a separate training example; the first 250 columns are the features, while the last column is has the real-valued labels.

Files:
train.txt - Training instances + labels (250 x 251)
test.txt - Test instances + labels (750 x 251)

Folders:
CrossValidation - contains the train and test data for each fold of the 5-fold cross-validation procedure on the training set.
